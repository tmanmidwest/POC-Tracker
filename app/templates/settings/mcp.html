Inbound — gateway access (e.g. Saviynt → MCP server)

Issue a separate bearer token per app or project that connects to the MCP server, and revoke any one of them independently. Until at least one token exists the MCP HTTP endpoint rejects every request (returns 503) — so it's safe to deploy the MCP container before configuring it here.
